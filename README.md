# 🍕 GCM-Pizzaria

Sistema web desenvolvido para gerenciamento de uma pizzaria, contendo uma API no backend e arquivos estáticos no frontend.

---

## 📌 Sobre o Projeto

O **GCM-Pizzaria** é uma aplicação web desenvolvida com foco em organização, separação de responsabilidades e boas práticas de desenvolvimento.

O projeto é dividido em:

- **Backend**: Responsável pela API e regras de negócio.
- **Frontend**: Responsável pela interface e arquivos estáticos.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- npm

---

## 📂 Estrutura do projeto

- GCM-Pizzaria/

- backend/ → API da aplicação (Node.js + Express)
- public/ → Arquivos estáticos (HTML, CSS, JS)
- package.json
- README.md
